# 세찬컴퍼니 인트라넷 V29.4 통합본

## 기준 버전
- V29.1 설치완료본 기준
- V29.2 / V29.3 미설치 요청사항을 하나로 통합

## 통합 반영 내용

### 1. 프리패스 명칭 변경
- 야근 적립 → 고객 추가 응대
- 휴무출근 적립 → 휴무 고객응대

### 2. 프리패스 적립 동의 팝업
- 회사 강제 지시/명령/사전 승인에 따른 연장근로·휴일근로 신청이 아님
- 본인의 자발적인 고객관리/판매활동/사후응대임
- 프리패스는 임금·수당이 아닌 복지제도 보상임
- 동의해야 적립 요청 가능

### 3. 동의 기록 저장
- 동의 여부
- 동의 시각
- 동의 문구
- 동의 스냅샷

### 4. 홈 중복 메뉴 제거
- 해피콜 바로가기
- 프리패스 바로가기
- 건의/문의 바로가기 삭제
- 모바일 하단 홈/해피콜/프리패스/건의/더보기 메뉴 삭제

### 5. 악세사리 상태변경 개선
- 기존 직접 타이핑 방식 제거
- 상태변경 클릭 시 팝업에서 상태 선택
- 주문 완료 선택 시 도착 예정일 날짜 선택
- 반품 선택 시 반품 사유 선택 및 기타 입력 가능

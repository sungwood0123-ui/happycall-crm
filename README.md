# 세찬 해피콜 관리시스템 V12 History Edit UI

## 수정
- 근무이력 입력을 한 줄 구조로 변경
  - 매장 / 직책 / 시작일 / ~ / 종료일 / 이력추가
- 매장 선택 박스 폭 축소
- 직책 박스 수평 정렬 개선
- 날짜 칸 폭 축소
- 근무이력 수정 기능 추가
  - 기존 이력의 수정 버튼 클릭
  - 위 입력창에 값 로드
  - 수정 저장 가능
- 근무이력 수정/추가/삭제 감사로그 유지
- 직원관리 메인 드롭다운 폭 균형 조정

## 기존 기능 검증
- 감사로그 컴포넌트 유지
- 직원별 현황 컴포넌트 유지
- 직원관리/근무이력 구조 유지
- 퇴사자 로그인 차단 유지
- happycall_targets delete 제거 유지
- npm build 검증

# 세찬 해피콜 관리시스템 V11.7 History Form Alignment

## 수정
- 근무이력 입력행 수평 불균형 수정
- 근무이력 입력 폼을 2줄 구조로 정리
  - 1줄: 매장 / 직책
  - 2줄: 시작일 / ~ / 종료일 / 이력 추가
- 입력칸/버튼 높이 40px 통일
- 날짜 박스 폭 통일
- ~ 표시 중앙 정렬
- 버튼 글자 중앙 정렬
- 글자 잘림/세로 나열 방지

## 기존 기능 검증
- 감사로그 컴포넌트 유지
- 직원별 현황 컴포넌트 유지
- 직원관리/근무이력 구조 유지
- 퇴사자 로그인 차단 유지
- happycall_targets delete 제거 유지
- npm build 검증

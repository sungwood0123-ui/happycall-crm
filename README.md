## 신규 반영
1. 오류보고 대시보드 미해결 카운트 보완
   - 완료
   - 해결
   - 해결완료
   - 처리완료
   - 조치완료
   - 종결
   - 닫힘
   상태는 미해결 카운트에서 제외

2. 직원관리 모바일 UI 최적화
   - 모바일에서 PC 테이블 축소 표시를 카드형 레이아웃으로 변경
   - 이름/매장/상태/권한/해피콜 배정/비밀번호/상세/저장을 세로 구조로 노출
   - 가로 스크롤과 드롭다운 잘림 최소화

3. 직원 추가 영역 개선
   - 기본 노출 대신 '+ 직원 추가' 버튼으로 접기/펼치기

4. 해피콜 배정 ON/OFF
   - 직원관리에서 직원별 해피콜 자동배정 ON/OFF 관리
   - 자동배정 OFF 직원은 해피콜 대상 생성/배정에서 제외

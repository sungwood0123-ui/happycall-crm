# 세찬컴퍼니 인트라넷 V29.47

## V29.47 변경 내용
- D+95·D+185는 통화불가 사유가 고객사정·마케팅 미동의·사고 발생건인 경우 다시 생성 대상에 포함
- 다른 통화불가 사유는 D+95·D+185에서도 제외
- 통화불가 상세 선택 목록에서 미성년자 제거
- 날짜 전용 계산을 보정하여 D+ 대상일이 하루 일찍 계산되는 오류 수정
- 사무실·에스플러스·퍼스트 고객 및 해당 매장 배정 건은 모든 해피콜 유형에서 생성 제외
- 기존 미처리 57건은 삭제하지 않고 생성 제외 처리했으며, 이미 처리된 과거 이력은 유지

## V29.46 변경 내용
- 통화결과가 정확히 `통화 불가`인 경우에만 통화 불가 고객으로 분류
- 부재중 및 통화 완료 고객이 통화 불가 고객으로 잘못 저장되던 오류 수정
- 기존 오분류 데이터는 운영 DB 검증 후 정상 대상에 복구

기준: V29.39 통합 수정본으로 복구

## 복구 내용
- V29.40 이후 추가된 해피콜 생성 진단 기능 및 관련 오류 코드를 제거
- 해피콜 생성 탭을 V29.39 통합 수정본 상태로 복원
- V29.39 통합 수정본에 포함된 팝업, 프리패스 승인 경로, 로딩 UI, 관리자 조정 및 직원관리 정렬 개선은 유지
- 실제 해피콜 생성 기준과 데이터는 변경하지 않음

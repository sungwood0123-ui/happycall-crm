# 세찬컴퍼니 인트라넷 V29.24

## 기준 버전
- V29.23 설치완료본 기준

## 반영 내용

### 1. 배정현황 기준 수정
- RAW 담당자 전체가 직원 목록에 뜨는 문제 수정
- 직원관리 테이블에 등록된 재직/퇴사자 중 실제 배정 고객 또는 배정 해피콜이 있는 사람만 표시
- 자동배정 또는 해피콜 생성이 한 번도 이루어지지 않은 과거 RAW 담당자는 배정현황에서 숨김
- 배정된 고객 탭은 RAW 판매자가 아니라 실제 시스템 배정 기준으로 표시

### 2. 건의/문의 불일치 수정
- 홈 대시보드 카운트와 건의/문의 탭 목록 조회 조건 일치
- 최고관리자/관리자는 전체 건의 확인
- 일반 직원은 본인 건의만 확인

### 3. 업데이트 공지 권한 수정
- 최고관리자는 모든 업데이트 내용 표시
- 관리자는 최고관리자 전용 업데이트 제외
- 점장/검수자/직원은 각 권한에 맞는 업데이트만 표시

### 4. 로딩 UX 개선
- 프리패스 승인 목록에 로딩 상태 표시
- 데이터 없음과 불러오는 중을 구분

# 세찬 해피콜 관리시스템 V11 Employee History/Menu

## 반영
- 상단 메뉴 카테고리 드롭다운 정리
- 직원관리 재직중/퇴사자 탭 분리
- 직원 입사일/퇴사일 관리
- 퇴사자 로그인 차단
- 직원 정보 최종저장 방식 적용
- 비밀번호 입력 + 1234 입력 버튼 묶음
- 근무이력 버튼/팝업 추가
- 근무이력 매장/직책/시작일/종료일 저장
- 종료일 공란은 현재 근무중 처리
- 근무이력 감사로그 기록

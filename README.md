# 세찬 해피콜 관리시스템 V4.1

V1 기능:
- 직원명 + 비밀번호 로그인
- 권한: 관리자 / 검수자 / 직원
- 직원관리
- 매장관리
- 내 해피콜 리스트
- 통화결과 저장
- 관리자 전체 현황

Vercel 환경변수:
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

## V3 추가 기능
- 해피콜 생성 탭
- 대상일 기준 D+1/D+7/D+13/D+95/D+185 생성

- 당월 D+ 대상 존재 시 월간 정기 해피콜 제외

- 기존 배정이력 유지
- 매장 재직자 자동배정
- happycall_targets 저장

## V3.1 수정 기능
- 직원관리에서 매장 직접입력 제거
- stores 테이블에 등록된 매장만 드롭다운 선택
- 폐점 매장은 목록에서 (폐점) 표시
- 매장명 오타로 인한 자동배정 오류 방지

## V3.2 수정 기능
- 직원관리 매장 선택 목록에 관리자 옵션 추가
- 관리자 옵션은 자동배정 제외 용도로 표시
- 관리자 계정이 특정 매장에 배정되어 해피콜 대상자로 잡히는 문제 방지

## V3.3 수정 기능
- 권한에 점장 추가
- 점장은 본인 해피콜 처리 가능
- 점장은 본인 매장 직원들의 해피콜 완료 여부 확인 가능
- 점장 매장 확인 화면은 수정 불가, 조회 전용

## V3.4 수정 기능
- 직원관리 신규 직원 생성 폼의 권한 드롭다운에도 점장 추가

## V3.5 수정 기능
- 신규 직원 추가 폼의 권한 드롭다운을 강제 수정
- 신규 추가 권한 목록: 직원 / 점장 / 검수자 / 관리자
- 수정 적용 블록 수: 1

## V3.6 긴급 수정

## V3.7 긴급 수정
